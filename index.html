The Question

Why are some countries rich and some poor?

One thing kept showing up in the data above everything else.

How corrupt the government is.

  • CPI score — 0 = extremely corrupt, 100 = very clean
  • GDP per capita — economic output per person, in USD

The Data

Two datasets. Both public. Both free.

Source Transparency International World Bank

Key column score — 0 (corrupt) to 100 (clean) gdp_pc — USD per person
Joined by iso3 country code + year → 178 countries, 2,067 rows

Reshaping the Data

# GDP file has years as columns — pivot_longer() fixes that
gdp_long <- gdp_raw |>
  select(`Country Name`, `Country Code`, `2012`:`2023`) |>
  pivot_longer(
    cols      = `2012`:`2023`,
    names_to  = "year",       # column names → "year" column
    values_to = "gdp_pc"      # values → "gdp_pc" column
  ) |>
  mutate(year = as.integer(year)) |>
  filter(!is.na(gdp_pc)) |>
  rename(iso3 = `Country Code`)

# inner_join() — only keeps countries in BOTH datasets
df <- inner_join(
  gdp_long,
  cpi_raw |> select(iso3, country, region, year, cpi = score),
  by = c("iso3", "year")
) |>
  filter(gdp_pc > 0) |>
  mutate(log_gdp = log10(gdp_pc))

cat("Countries:", n_distinct(df$iso3), "| Rows:", nrow(df))

So How Strong Is This Really?

We ran one line of R on 178 countries.

lm(log_gdp ~ cpi, data = df)

Out of every reason a country could be rich or poor — geography, history, trade, education, resources —

corruption score alone accounts for 62.3% of it.

And every single point a country cleans up —

6.3% more money in the pocket of the average person.

That’s schools getting funded. Hospitals getting built. Kids getting a shot they wouldn’t have had.

In 2024 — Acemoglu, Johnson & Robinson won the Nobel Prize in Economics.

A lifetime of research. Their conclusion —

“What separates rich countries from poor countries is whether institutions work for the people — or extract from them.”

And our data — 178 countries, 12 years — points exactly in the same direction.

What We Found

  • 178 countries, 12 years of real data
  • CPI alone explains 62% of why some countries are rich and others aren’t
  • Very clean countries are 27× richer than very corrupt ones
  • The cause is still open — does corruption create poverty, or does poverty create corruption?
